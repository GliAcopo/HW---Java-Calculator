# Application Options
To choose the theme for the app, pass it as the first argument of the main.
```java
java -jar app.jar light
```
or
```java
java -jar app.jar dark
```

light is the default theme.

To press the buttons, you can also use the corresponding keyboard keys as accelerators.
ESC = "AC";
ENTER = "=";
ALT + BUTTON TEXT = the relevant button.
